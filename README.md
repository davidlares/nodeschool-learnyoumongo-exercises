## LearnYouMongo

Basic exercises on the main features and functions of MongoDB (NoSQL Database Engine).

This repository is part of the NodeSchool Workshops program (LearnYouMongo).

# Tools

	- NodeJs
	- MongoDB (version 3)
	- Mongod Daemon running
